## Your First Playwright Script

Let's create a simple script that opens a browser, navigates to the Playwright website, clicks a link, and prints the resulting URL.

<Steps>
  <Step title="Create a Python File">
    Create a new file called `app.py` in your project directory:

    ```bash
    touch app.py
    ```
  </Step>

  <Step title="Write the Automation Script">
    Open `app.py` and add the following code:

    ```python app.py
    from playwright.sync_api import sync_playwright


    with sync_playwright() as playwright:
        # Launch a browser
        browser = playwright.chromium.launch(headless=False, slow_mo=500)
        # Create a new page
        page = browser.new_page()
        # Visit the playwright website
        page.goto("https://playwright.dev/python")

        # Locate a link element with "Docs" text
        docs_button = page.get_by_role('link', name="Docs")
        docs_button.click()

        # Get the url
        print("Docs:", page.url)

        browser.close()
    ```

    <Note>
      We're using `headless=False` to see the browser in action, and `slow_mo=500` to slow down operations by 500ms so you can observe what's happening.
    </Note>
  </Step>

  <Step title="Run the Script">
    Execute your script:

    ```bash
    python app.py
    ```

    You should see:
    1. A Chromium browser window opens
    2. The browser navigates to the Playwright Python homepage
    3. The "Docs" link is clicked
    4. The terminal prints the resulting URL
    5. The browser closes
  </Step>
</Steps>

## Understanding the Code

Let's break down what each part does:

### Context Manager

```python
with sync_playwright() as playwright:
```

The `sync_playwright()` context manager initializes the Playwright environment. It automatically handles setup and cleanup.

<Tip>
  Using a context manager (`with` statement) ensures resources are properly cleaned up, even if an error occurs.
</Tip>

### Browser Launch

```python
browser = playwright.chromium.launch(headless=False, slow_mo=500)
```

- `playwright.chromium` - Launches Chromium (you can also use `firefox` or `webkit`)
- `headless=False` - Shows the browser window (set to `True` for headless mode)
- `slow_mo=500` - Adds 500ms delay between operations for visibility

### Create a Page

```python
page = browser.new_page()
```

Creates a new browser tab/page where actions will be performed.

### Navigate

```python
page.goto("https://playwright.dev/python")
```

Navigates to the specified URL. Playwright automatically waits for the page to load.

### Locate and Interact

```python
docs_button = page.get_by_role('link', name="Docs")
docs_button.click()
```

- `get_by_role('link', name="Docs")` - Finds a link element with accessible name "Docs"
- `click()` - Clicks the element (waits for it to be clickable first)

<Note>
  Playwright automatically waits for elements to be ready before performing actions. No manual waits needed!
</Note>

### Get Information

```python
print("Docs:", page.url)
```

Retrieves and prints the current page URL after navigation.

## Trying Different Browsers

Playwright supports multiple browser engines. Try running your script with different browsers:

<CodeGroup>
```python Chromium
browser = playwright.chromium.launch(headless=False, slow_mo=500)
```

```python Firefox
browser = playwright.firefox.launch(headless=False, slow_mo=500)
```

```python WebKit (Safari)
browser = playwright.webkit.launch(headless=False, slow_mo=500)
```
</CodeGroup>

## Running in Headless Mode

For production or CI/CD environments, run in headless mode (no visible browser):

```python
browser = playwright.chromium.launch(headless=True)
```

Or simply omit the parameter (headless is the default):

```python
browser = playwright.chromium.launch()
```

## Common Operations

Here are some common operations you can perform:

### Typing Text

```python
page.get_by_label("Email").fill("user@example.com")
```

### Selecting Options

```python
page.get_by_label("Country").select_option("United States")
```

### Checking Checkboxes

```python
page.get_by_label("I agree to terms").check()
```

### Taking Screenshots

```python
page.screenshot(path="screenshot.png")
```

### Waiting for Elements

```python
page.wait_for_selector("text=Welcome")
```

<Tip>
  Most Playwright actions have built-in auto-waiting, so explicit waits are rarely needed.
</Tip>

## Enhanced Example

Here's a more complete example that demonstrates multiple interactions:

```python enhanced_app.py
from playwright.sync_api import sync_playwright


with sync_playwright() as playwright:
    browser = playwright.chromium.launch(headless=False, slow_mo=500)
    page = browser.new_page()
    
    # Navigate to the Playwright Python homepage
    page.goto("https://playwright.dev/python")
    print(f"Page title: {page.title()}")
    
    # Click the "GET STARTED" link
    get_started = page.get_by_role('link', name="GET STARTED")
    get_started.click()
    
    print(f"Navigated to: {page.url}")
    
    # Take a screenshot
    page.screenshot(path="getting-started.png")
    print("Screenshot saved as getting-started.png")
    
    browser.close()
```

## Troubleshooting

### Browser Not Launching

If the browser doesn't launch, ensure browsers are installed:

```bash
playwright install
```

### Element Not Found

If an element isn't found, the page might still be loading. Playwright usually handles this automatically, but you can add explicit waits:

```python
page.wait_for_load_state("networkidle")
```

### Timeout Errors

Increase the default timeout if needed:

```python
page.set_default_timeout(60000)  # 60 seconds
```
